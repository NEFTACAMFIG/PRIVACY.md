Privacy Policy for Data Entry Automation for JMA

This application is an internal tool designed to integrate with QuickBooks Online.

1. Data Collection: We only process data provided by the user via CSV files (vendor names, amounts, and account IDs) and the necessary OAuth2 tokens for API authentication.

2. Data Usage: Data is used exclusively to create "Purchase" records in the user's QuickBooks account.

3. Data Sharing: We do not sell, trade, or share user data with third parties.

4. Storage: All data processing occurs locally on the user's machine. Tokens are stored securely and used only for API communication.
